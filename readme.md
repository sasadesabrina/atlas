# 🧭 Atlas — Seu Guia no Universo dos Games

O **Atlas** é uma plataforma acolhedora de descoberta e organização de jogos criada para ajudar jogadores a explorar novos mundos, registrar suas experiências e construir sua própria jornada através do universo dos games. 

Diferente de lojas digitais focadas apenas em transações, o Atlas funciona como um mapa afetivo focado na descoberta, exploração e planejamento pessoal de forma visual e inspiradora. Porque nós não procuramos apenas produtos para comprar; procuramos histórias e aventuras para viver! ✨

---

## 🌸 O Problema & A Nossa Solução

* **O Problema:** Hoje, existem milhares de jogos espalhados por diversas lojas digitais. É muito fácil comprar, mas é difícil descobrir novas experiências que combinem com os nossos gostos ou manter uma organização clara do que realmente queremos jogar. Listas improvisadas ou anotações perdidas acabam tirando a magia do planejamento.
* **A Solução:** O Atlas centraliza a exploração. Com mecanismos de busca inteligentes, filtros e uma biblioteca altamente personalizada, transformamos a organização dos seus jogos em uma experiência leve, bonita e intuitiva.

---

## 🎒 Recursos Principais (Features)

* **🗺️ Modo Exploração:** Pesquise títulos e descubra novas aventuras com detalhes profundos sobre cada universo.
* **📦 Organização por Diários de Bordo:** Separe seus jogos de forma fofa através de categorias que refletem o seu progresso:
  * 🌟 *Quero Explorar* — Seu mural de desejos e próximas paradas.
  * ⚔️ *Em Jornada* — Os mundos que você está desbravando no momento.
  * 🏆 *Conquistados* — Suas vitórias e jornadas finalizadas.
  * 💖 *Favoritos* — Aqueles jogos que deixaram uma marca eterna no seu coração.
* **🎨 Interface imersiva:** Identidade visual inspirada em mapas celestiais e elementos de navegação, projetada para ser um porto seguro para os seus registros de jogos.

---

## 🛠️ Tecnologias Utilizadas (Tech Stack)

Este projeto foi construído utilizando as ferramentas mais modernas do ecossistema web para garantir uma experiência rápida, fluida e reativa:

* **Framework:** [Next.js 16](https://nextjs.org/) (App Router) — Renderização ágil e roteamento otimizado de última geração.
* **Biblioteca:** [React 19](https://react.dev) — Aproveitando os recursos mais recentes de concorrência e gerenciamento de estado.
* **Estilização:** [Tailwind CSS v4](https://tailwindcss.com/) — Design system moderno configurado diretamente via CSS-first, garantindo uma interface limpa e tempos de build incrivelmente rápidos.

## 🚀 Como Rodar o Projeto Localmente

Siga os passos abaixo para erguer o seu próprio mapa do Atlas na sua máquina:

### 1. Clonar o repositório
```bash
git clone https://github.com
cd atlas
```

### 2. Instalar as dependências
```bash
npm install
# ou com yarn/pnpm/bun
```

### 3. Rodar o servidor de desenvolvimento
```bash
npm run dev
# ou yarn dev / pnpm dev / bun dev
```

Abra o seu navegador em [http://localhost:3000](http://localhost:3000) para ver o Atlas ganhando vida! 🚀

## 📂 Estrutura do Projeto

Uma visão geral de como o ecossistema do Atlas está organizado:

```text
atlas/
├── app/                  # Roteamento baseado em App Router (Next.js 16)
│   ├── layout.tsx        # Layout global e provedores
│   ├── page.tsx          # Página principal (Dashboard de exploração)
│   └── components/       # Componentes visuais e reutilizáveis (React 19)
├── public/               # Ativos estáticos (ícones místicos, imagens)
└── app/global.css        # Configurações e tokens de estilo (Tailwind v4)
```
